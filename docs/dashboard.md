# ログイン後画面 仕様書（UI / 機能仕様）

## 1. 画面概要

この画面は、ユーザーがログイン後に自身の卓（テーブル）スケジュールや他ユーザーの卓情報を確認できるメインページです。
卓の公開・非公開状態や所属情報を視覚的に確認でき、参加・閲覧・操作が可能です。

## 2. 画面構成

### 2.1 ヘッダー部

| 要素 | 内容 |
|------|------|
| 左上アイコン | ユーザーのプロフィール画像またはアイコン |
| ユーザー名表示 | ログイン中のユーザー名（例：「卓日程くん」） |
| 右上メニュー | ハンバーガーメニュー（3つのボタンが縦に並ぶ）<br>メニュー押下で設定・ログアウト・ヘルプなどへ遷移 |

### 2.2 メインコンテンツ

#### ① あなたの卓予定ボタン

- **ボタン名**：「あなたの卓予定」
- **機能**：自分が参加予定または主催している卓一覧を表示
- **遷移先**：ユーザー個別のスケジュール画面

#### ② 卓一覧表示エリア

卓（テーブル）をカード形式で表示

各卓カードは以下の情報を含む

| 項目 | 内容 |
|------|------|
| アイコン枠 | 卓のアイコンまたは主催者のアバター画像 |
| 卓名 | 「卓1」「Aさん卓」など |
| 公開ステータス | 公開卓：地球アイコンなし<br>身内卓（非公開）：🔒ロックアイコン表示 |
| 時間帯 | 「21時〜24時の卓です」固定文言（将来的には可変可能） |

#### ③ 卓の種類

| 種別 | 表示例 | 公開範囲 |
|------|--------|----------|
| 公開卓 | 卓2、Cさん卓など | すべてのユーザーが閲覧可能 |
| 身内卓 | 卓1、Aさん卓、Bさん卓など（🔒アイコン付） | 承認されたメンバーのみ参加可能 |

## 3. 機能仕様

### 3.1 卓カードの操作

| 操作 | 内容 |
|------|------|
| クリック | 卓の詳細ページへ遷移（詳細・参加・チャットなど） |
| 長押しまたは右クリック | （オプション）詳細メニューを表示予定（削除・編集など） |

### 3.2 ハンバーガーメニュー

| メニュー項目 | 機能 |
|-------------|------|
| 設定 | プロフィール・通知設定 |
| ログアウト | セッション終了 |
| ヘルプ | サイト説明・操作ガイド |

## 4. スタイル仕様（デザイン）

| 要素 | スタイル概要 |
|------|-------------|
| 全体背景 | #0D0D0D（黒系） |
| テキスト色 | 白（#FFFFFF） |
| カード枠線 | 白またはグレー系（#CCCCCC） |
| ロックアイコン | オレンジ（#FFA500） |
| フォント | 日本語UI向けサンセリフ体（例：Noto Sans JP） |
| レイアウト | グリッド形式（3列程度）で卓カードを整列 |
| レスポンシブ対応 | スマホ・タブレットでは1〜2列表示に切り替え |

## 5. 遷移図（概要）

```
[ログイン画面]
   ↓
[ログイン後画面]
   ├─ あなたの卓予定 → [スケジュール画面]
   ├─ 卓カード選択 → [卓詳細ページ]
   └─ ハンバーガーメニュー → [みんなの卓 / あなたの卓 / ログアウト]
```

## 6. 技術仕様

### 6.1 必要なモデル

- **User**: ユーザー情報
- **Table**: 卓情報（卓名、公開状態、主催者など）
- **TableMember**: 卓メンバー情報（参加者・承認状態など）

### 6.2 必要なコントローラー

- **DashboardController**: メイン画面の表示
- **TablesController**: 卓の詳細・操作
- **SchedulesController**: ユーザーのスケジュール管理

### 6.3 必要なビュー

- **dashboard/index.html.erb**: メイン画面
- **shared/_header.html.erb**: ヘッダー部
- **shared/_table_card.html.erb**: 卓カード部分テンプレート
